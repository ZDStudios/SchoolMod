// Setup

Getting Started

SchoolMod is still in early development. Here's how to run it from source right now.

1

Fork & Clone

Fork the repo on GitHub, then clone it to your local machine.

2

Navigate Into the Project

Open the project folder in your terminal or code editor.

3

Load & Test

Follow the setup guide in the /docs folder once available.

4

Packaged Release Coming Soon

A browser extension build is planned. Watch the repo to get notified.

bash
# Clone the repository git clone https://github.com/zdstudios/SchoolMod.git   # Navigate into the project cd SchoolMod   # Follow docs for setup cat docs/SETUP.md
