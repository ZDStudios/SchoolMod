# BetterKahoot+

A first-pass Kahoot enhancement module for SchoolMod.

## Included in this initial version

- Floating **BetterKahoot+** button
- Compact quick panel
- **Focus mode** for stronger visual emphasis on interactive elements
- **Zen chrome** mode to reduce visual clutter
- Safe, cosmetic-only implementation for the first release

## File

- `BetterKahoot+.js`

## Notes

This is an intentionally safe starting point. It does **not** automate gameplay or bypass platform rules. It only adds lightweight quality-of-life UI improvements.
